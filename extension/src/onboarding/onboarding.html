vnsh
The Ephemeral Dropbox for AI
01

Share anything, encrypted

vnsh encrypts your data before it leaves your browser. The server never sees your content. Links auto-expire in 24 hours.

Your text → AES-256 Encrypt → Upload blob → Shareable link
02

Try it: Right-click Share

Select the text below, then right-click and choose "Share via vnsh".

const server = http.createServer((req, res) => { console.log(`${req.method} ${req.url}`); res.writeHead(200, { 'Content-Type': 'text/plain' }); res.end('Hello from vnsh!'); }); server.listen(3000);

The encrypted link is automatically copied to your clipboard.

03

AI Debug Bundle

The killer feature: one click to package your current page's debug context (screenshot + console errors + selected text) into a single encrypted link. Paste it to any AI assistant.

Cmd + Shift + D Debug Bundle
Cmd + Shift + S Screenshot Share
04

You're all set!

Here's what you can do:

  • Right-click selected text → Share via vnsh
  • Right-click an image → Share image via vnsh
  • Right-click → AI Debug Bundle for full context
  • Right-click → Save to vnsh (local clipboard)
  • Click the extension icon for the popup panel
Privacy promise: All encryption happens in your browser. The vnsh server is mathematically blind to your data.
